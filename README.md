## 贡献者须知

在你做出贡献前，请知悉：
- 本书不是原书的续集，只对原题集内容做重新整理与排版，讲义部分重写。它与原版不存在接续关系。
- 本书使用 Typst 重制，不接受 Word, Markdown, LaTeX 等形式的贡献。
- 本书重制版将采用 CC-0 协议发布。你对本书的贡献也会以 CC-0 协议发布。我会在书内为主要贡献者署名；但不限制后续使用者传播、改变署名及商用。
- 本书所有源代码将公开于 [GitHub 仓库](https://github.com/cppHusky/IntegralCollection)当中，并在完成重制后，可能进行长期维护。
